Health Calculator Application - Setup and Run Instructions

Frontend Setup:
1. Ensure you have Node.js and npm installed on your machine. You can download them from https://nodejs.org/en/download/ if you haven't already.
2. Open a terminal or command prompt.
3. Navigate to the root directory of the frontend (where the index.html and script.js files are located).
4. Install any required dependencies by running the following command:
   npm install
5. Once the dependencies are installed, start the frontend server using the following command:
   npm start
6. The frontend should now be running on http://localhost:3000. Open your web browser and visit http://localhost:3000 to access the Health Calculator.

Backend Setup:
1. Ensure you have Node.js installed on your machine. You can download it from https://nodejs.org/en/download/ if you haven't already.
2. Open a terminal or command prompt.
3. Navigate to the root directory of the backend (where the index.js and package.json files are located).
4. Install any required dependencies by running the following command:
   npm install
5. Once the dependencies are installed, start the backend server using the following command:
   node index.js
6. The backend should now be running on http://localhost:3000. It is ready to handle API requests from the frontend.

Running the Application:
1. Ensure both the frontend and backend servers are running.
2. Open your web browser and visit http://localhost:3000 to access the Health Calculator.
3. You can now use the Health Calculator to calculate BMI, body fat percentage, ideal weight, and calories burned. The results will be displayed on the web page.

Note: rarely when you directly click on the link, you may not see calculate for all the functionalities, but it always works when you want to calculate them through the live server.
